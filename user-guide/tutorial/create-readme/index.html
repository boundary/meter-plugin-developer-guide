Add README

Each meter plugin includes a file named README.md that is displayed in the Plugin Configuration dialog that provides documentation regarding the plugins requirements, setup instructions, configuration parameters, metrics, dashboards, etc. needed for the operation of the meter plugin.

Content that needs to be included in a README is discussed in Documentation.

Adding README.md

Add the README.md file to your repository by the running the following:

$ git checkout step6
